⚠️ Global Response Envelope (Xiaoliang / Leon's implementation) All Auth, Courses, and Materials endpoints wrap their response in an ApiResponse object. Frontend must read response.data.data (axios adds one .data layer, ApiResponse adds another). 
{
  "success": true,
  "message": "...",
  "data": { /* actual payload */ }
}

// Error shape (all 4xx / 5xx):
{
  "success": false,
  "message": "Error description",
  "data": null
}

// HTTP status codes: 400 validation | 401 wrong credentials |
// 403 wrong role/owner | 404 not found | 409 conflict | 500 server error
